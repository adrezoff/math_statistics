# math_statistics

Единый собранный PDF файл —`math_statistics.pdf`, его можно создать/обновить при помощи скрипта `make_pdf.py`, **отдельным Pull Request** в main.

PS: `DejaVuLGCSans.ttf` — шрифт для автооглавления.

## Как залить изменения?
1) Создаем ветку (от `main`) с указанием билета который редактируем.
2) Добавляем тело билета в md файл.
3) Конвертируем `md` в `pdf`, с точно **таким же именем** файла.

## Как работать с Pull Request(PR)?
1) Если это первый push с изменениями, то зайти на [github](https://github.com/adrezoff/math_statistics/pulls) и открыть PR.
2) Если вы запушили все изменения по билету, то отправляем PR на ревью.
3) Замерджить в `main` можно только после 2х аппрувов от других людей, и только тому кто открыл этот PR.
