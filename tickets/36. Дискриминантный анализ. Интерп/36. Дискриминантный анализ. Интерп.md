# 36. Дискриминантный анализ. Интерпретация коэффициентов (нестандартизованных, стандартизованных, структурных) канонических дискриминантных функций.

**Дискриминантный анализ** — это статистический метод, предназначенный для решения задач классификации, то есть
отнесения объектов к одной из нескольких групп на основе набора признаков (предикторов).

## 1. Канонические дискриминантные функции

Каноническая дискриминантная функция (КДФ) представляет собой линейную комбинацию исходных переменных (предикторов),
которая максимизирует различия между группами. В общем виде функция записывается как:

$$d = c + b_1 X_1 + b_2 X_2 + \dots + b_p X_p$$

Где:

* $d$ — значение (счет) дискриминантной функции;
* $c$ — константа;
* $b_i$ — коэффициенты дискриминации;
* $X_i$ — значения независимых переменных.

## 2. Интерпретация коэффициентов КДФ

Для понимания вклада каждой переменной в разделение групп используются три вида коэффициентов: нестандартизованные,
стандартизованные и структурные.

### А. Нестандартизованные коэффициенты

Эти коэффициенты применяются непосредственно к исходным (сырым) значениям переменных.

* **Смысл:** Используются главным образом для вычисления дискриминантных меток (счетов) объектов и определения положения
  центроидов групп в многомерном пространстве.
* **Ограничение:** Их нельзя использовать для оценки относительной важности переменных, так как они зависят от единиц
  измерения (например, если одна переменная измеряется в миллионах, а другая в процентах, их коэффициенты будут
  несопоставимы).

### Б. Стандартизованные коэффициенты

Для получения этих коэффициентов переменные предварительно центрируются и нормируются.

* **Смысл:** Они показывают относительный вклад каждой переменной в дискриминантную функцию при условии, что все
  переменные приведены к единому масштабу.
* **Интерпретация:** Чем выше абсолютное значение стандартизованного коэффициента, тем больший вклад вносит данная
  переменная в расчет значения функции. Это позволяет ранжировать переменные по их значимости для дискриминации.

### В. Структурные коэффициенты (структурная матрица)

Структурные коэффициенты представляют собой корреляции между исходными переменными и канонической дискриминантной
функцией.

* **Смысл:** Они отражают «чистую» связь переменной с функцией. В отличие от стандартизованных коэффициентов, на
  структурные коэффициенты не влияет мультиколлинеарность (взаимозависимость) между предикторами.
* **Интерпретация:** * Если значение близко к $\pm 1$, переменная сильно связана с данной функцией.
    * Переменные с высокими структурными коэффициентами используются для содержательного именования (интерпретации)
      функции (например, если с функцией сильно коррелируют финансовые показатели, её называют «фактором финансовой
      стабильности»).
* **Важное отличие (дополнено автором):** Стандартизованные коэффициенты показывают вклад переменной в *вычисление*
  функции, а структурные — степень *связи* переменной с функцией.

## 3. Показатели качества и вывода

Для оценки качества функций в пособии приводятся следующие характеристики:

1. **Собственное значение (Eigenvalue):** Отражает долю вариации зависимой переменной (групповой принадлежности),
   объясненную данной функцией. Чем оно выше, тем лучше функция разделяет группы.
2. **Каноническая корреляция:** Мера связи между дискриминантными счетами и группами. Квадрат этого коэффициента
   показывает долю дисперсии функции, связанную с различиями между группами.
3. **Лямбда Уилкса ($\Lambda$):** Показывает долю общей изменчивости, не объясненную групповыми различиями. Значения
   близки к 0 свидетельствуют о хорошей дискриминации, значения близки к 1 — о плохой.
4. **Центроиды групп:** Средние значения дискриминантных функций для каждой группы. Группы максимально различимы, если
   их центроиды значительно удалены друг от друга.
