# Билет: Генеральная и выборочная дисперсия. Оценка параметров.

## 1. Понятия генеральной и выборочной дисперсии

### Генеральная дисперсия ($D_г$)
**Генеральная дисперсия** — это среднее арифметическое квадратов отклонений значений генеральной совокупности от их среднего значения (генерального среднего $\bar{x}_г$). Она характеризует рассеяние признака во всей изучаемой совокупности.

Если генеральная совокупность имеет объем $N$:
$$D_г = \frac{\sum_{i=1}^{N} (x_i - \bar{x}_г)^2}{N}$$

### Выборочная дисперсия ($D_в$ или $\hat{D}$)
**Выборочная дисперсия** — это среднее арифметическое квадратов отклонений наблюдаемых значений выборки от их выборочного среднего $\bar{x}_в$.

Для выборки объема $n$:
$$D_в = \frac{\sum_{i=1}^{n} (x_i - \bar{x}_в)^2}{n}$$

Если данные представлены в виде статистического ряда (с частотами $n_i$):
$$D_в = \frac{\sum n_i (x_i - \bar{x}_в)^2}{n} = \sum \frac{n_i}{n} (x_i - \bar{x}_в)^2$$

---

## 2. Оценка генеральной дисперсии по выборочной

Выборочная дисперсия $D_в$ является **точечной оценкой** генеральной дисперсии. Однако, согласно теории оценок, она является **смещенной** оценкой. 
Математическое ожидание выборочной дисперсии меньше генеральной дисперсии:
$$M(D_в) = \frac{n-1}{n} D_г$$

Это означает, что при использовании $D_в$ для оценки $D_г$ мы будем систематически получать заниженный результат, особенно при малых объемах выборки ($n$).

---

## 3. Исправленная выборочная дисперсия ($s^2$)

Чтобы устранить смещение, вводится понятие **исправленной выборочной дисперсии**. Она обозначается $s^2$ и вычисляется по формуле:
$$s^2 = \frac{n}{n-1} D_в = \frac{\sum_{i=1}^{n} (x_i - \bar{x}_в)^2}{n-1}$$

**Свойство несмещенности:**
Исправленная дисперсия является несмещенной оценкой генеральной дисперсии:
$$M(s^2) = D_г$$

---

## 4. Свойства выборочной дисперсии (с доказательством)

### Свойство 1: Расчетная формула
Дисперсия равна разности между средним квадратов значений и квадратом их среднего.
$$D_в = \overline{x^2} - (\bar{x}_в)^2$$

**Доказательство:**
$$D_в = \frac{1}{n} \sum (x_i - \bar{x})^2 = \frac{1}{n} \sum (x_i^2 - 2x_i\bar{x} + \bar{x}^2) =$$
$$= \frac{1}{n} \sum x_i^2 - 2\bar{x} \left( \frac{1}{n} \sum x_i \right) + \frac{1}{n} \sum \bar{x}^2 =$$
$$= \overline{x^2} - 2\bar{x} \cdot \bar{x} + \bar{x}^2 = \overline{x^2} - \bar{x}^2$$

### Свойство 2: Прибавление константы
Если ко всем значениям выборки прибавить постоянную величину $C$, то дисперсия не изменится: $D(X + C) = D(X)$.

**Доказательство:**
Новые значения $y_i = x_i + C$, тогда $\bar{y} = \bar{x} + C$.
$$D_y = \frac{1}{n} \sum (y_i - \bar{y})^2 = \frac{1}{n} \sum ((x_i + C) - (\bar{x} + C))^2 =$$
$$= \frac{1}{n} \sum (x_i - \bar{x})^2 = D_x$$

### Свойство 3: Умножение на константу
Если все значения выборки умножить на постоянную величину $k$, то дисперсия увеличится в $k^2$ раз: $D(kX) = k^2 D(X)$.

**Доказательство:**
Новые значения $y_i = kx_i$, тогда $\bar{y} = k\bar{x}$.
$$D_y = \frac{1}{n} \sum (kx_i - k\bar{x})^2 = \frac{1}{n} \sum k^2(x_i - \bar{x})^2 = k^2 D_x$$

### Свойство 4: Минимальное свойство дисперсии
Средний квадрат отклонений значений выборки минимален, если отклонения считаются от среднего арифметического.
$$\sum (x_i - C)^2 > \sum (x_i - \bar{x})^2 \text{ при любом } C \neq \bar{x}$$

---

## 5. Резюме
1. **Выборочная дисперсия ($D_в$)** — смещенная оценка.
2. **Исправленная дисперсия ($s^2$)** — несмещенная оценка, получается путем умножения $D_в$ на коэффициент «поправки Бесселя» $\frac{n}{n-1}$.
3. При больших объемах выборки ($n \to \infty$) разница между $D_в$ и $s^2$ исчезает.
