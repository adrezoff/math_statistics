# Билет №12. Проверка гипотезы о законе распределения генеральной совокупности

## 1. Понятие статистической гипотезы о законе распределения
**Статистической гипотезой** называется предположение о виде неизвестного закона распределения генеральной совокупности или о параметрах известных законов, проверяемое на основании выборки.

Гипотезы о законе распределения относятся к **непараметрическим**, так как в них проверяется предположение о виде функции распределения $F(x)$ в целом, а не только о её параметрах.

* **Нулевая гипотеза ($H_0$):** Генеральная совокупность распределена по закону $F_0(x)$.
* **Альтернативная гипотеза ($H_1$):** Генеральная совокупность распределена по иному закону.

Для проверки таких гипотез используются специальные статистики, называемые **критериями согласия**.

---

## 2. Критерий согласия Пирсона (Хи-квадрат, $\chi^2$)
Критерий Пирсона является наиболее универсальным и часто используемым критерием для проверки гипотез о распределении.

### Алгоритм проверки:
1.  **Группировка данных:** Выборка объема $n$ разбивается на $k$ интервалов (групп).
2.  **Подсчет частот:** Для каждого интервала вычисляется:
    * $n_i$ — **эмпирическая частота** (число вариант, попавших в $i$-й интервал).
    * $P_i$ — **теоретическая вероятность** попадания случайной величины в $i$-й интервал, вычисляемая на основе гипотетического закона $F_0(x)$.
    * $n P_i = n'_i$ — **теоретическая частота** (сколько вариант *должно было* попасть в интервал в среднем).

3.  **Вычисление статистики критерия:**
    Статистика Пирсона характеризует расхождение между эмпирическими и теоретическими частотами:
    $$\chi^2_{набл} = \sum_{i=1}^{k} \frac{(n_i - n P_i)^2}{n P_i}$$

### Обоснование и распределение:
Согласно теореме Пирсона, при $n \to \infty$ данная статистика стремится к **распределению $\chi^2$** (хи-квадрат).
Количество **степеней свободы** вычисляется по формуле:
$$df = k - r - 1$$
где:
* $k$ — число интервалов;
* $r$ — число параметров закона распределения, оцененных по данной выборке (например, для нормального закона $r=2$: среднее и дисперсия).

### Принятие решения:
1.  Задается уровень значимости $\alpha$.
2.  По таблице распределения $\chi^2$ находится критическое значение $\chi^2_{крит}(\alpha, df)$.
3.  **Правило:**
    * Если $\chi^2_{набл} < \chi^2_{крит}$, то гипотеза $H_0$ **принимается** (не противоречит данным).
    * Если $\chi^2_{набл} > \chi^2_{крит}$, то гипотеза $H_0$ **отвергается**.

## 3. Критерий согласия Колмогорова ($\lambda$)
Критерий Колмогорова применяется для проверки гипотез о непрерывных распределениях с полностью известными параметрами.

### Сущность критерия:
Критерий основан на сравнении эмпирической функции распределения $F^*_n(x)$ и гипотетической функции распределения $F_0(x)$. В качестве меры расхождения берется максимальное значение модуля их разности:
$$D_n = \max |F^*_n(x) - F_0(x)|$$

### Статистика Колмогорова:
Колмогоров доказал, что для любой непрерывной функции $F_0(x)$ при $n \to \infty$ случайная величина:
$$\lambda = D_n \sqrt{n}$$
подчиняется **закону распределения Колмогорова**.

### Принятие решения:
1.  Вычисляется наблюдаемое значение $\lambda_{набл} = D_{max} \sqrt{n}$.
2.  По таблице распределения Колмогорова находится критическое значение $\lambda_{\alpha}$ для выбранного уровня значимости (например, для $\alpha = 0.05$, $\lambda_{0.05} = 1.36$).
3.  **Правило:**
    * Если $\lambda_{набл} < \lambda_{\alpha}$, гипотеза $H_0$ **не отклоняется**.
    * Если $\lambda_{набл} \ge \lambda_{\alpha}$, гипотеза $H_0$ **отвергается**.

---

## 4. Сравнение критериев

| Характеристика | Критерий Пирсона ($\chi^2$) | Критерий Колмогорова ($\lambda$) |
| :--- | :--- | :--- |
| **Тип данных** | Сгруппированные (дискретные и интервальные) | Непрерывные несгруппированные |
| **Параметры** | Позволяет оценивать параметры по выборке (меняя $df$) | Параметры распределения должны быть известны заранее |
| **Объем выборки** | Требует больших выборок ($n \ge 50$, $n_i \ge 5$) | Более чувствителен при небольших $n$, если закон известен точно |
| **Сложность** | Требует разбиения на интервалы | Требует нахождения $D_{max}$ по всей выборке |
